# Projeto: De Pátria para Pátria — Galeria de Arte TripleTen

Uma página web responsiva e adaptável desenvolvida como parte do currículo de Desenvolvimento Web da TripleTen. O projeto consiste em uma galeria que reúne histórias e fotografias de profissionais de tecnologia de diversos cantos do mundo.

---

## 🚀 Tecnologias Utilizadas

* **HTML5:** Estruturação semântica e acessível da página.
* **CSS3:** Estilização baseada na metodologia BEM (Block, Element, Modifier).
* **Responsividade:** Uso de Media Queries para adaptação perfeita em Desktop (1280px), Tablet (768px) e Celular (320px).

---

## 🎨 Funcionalidades e Aprendizados

* **Metodologia BEM:** Organização do código CSS por blocos independentes (`header`, `lead`, `intro`, `footer`), facilitando a manutenção.
* **Fidelidade ao Design (Figma):** Tradução exata de tipografia, larguras máximas, espaçamentos e resoluções responsivas diretamente do layout do designer para o código.
* **Layout Fluido:** Otimização de imagens e blocos de texto para evitar barras de rolagem horizontal em telas menores.

---

## 📂 Como visualizar o projeto localmente

1. Faça o clone deste repositório para a sua máquina.
2. Abra a pasta do projeto no seu editor de código (ex: VS Code).
3. Execute o arquivo `index.html` utilizando a extensão **Live Server** para visualizar a página no seu navegador.
